# Emacs Config Dependencies

- `use-package` — package management convenience
- `jinx` — spell checker (requires system library `enchant`)
- `pdf-tools` — PDF viewing inside Emacs
- `auctex` — LaTeX editing
- `haskell-mode` — Haskell editing
- `lsp-mode` — Language Server Protocol client
- `magit-section` — used by `magit`, but also available standalone
- `gptel` — LLM client (OpenAI, Gemini, etc.)
- `airline-themes` — status line themes
- `lean4-mode` — Lean 4 support (install manually from GitHub)
